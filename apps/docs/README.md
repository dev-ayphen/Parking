# ParkSwift Documentation

Complete documentation for the ParkSwift parking management platform.

## 📚 Documentation Files

- **API.md** - REST API endpoints and documentation
- **FLOWS.md** - User flows and business logic
- **ARCHITECTURE.md** - System architecture and design

## 🏗️ Architecture

ParkSwift is built with a monorepo structure using pnpm and Nx:

- **Mobile**: React Native + Expo
- **Web**: Next.js + React
- **API**: Node.js + Express
- **Database**: PostgreSQL + Prisma
- **Shared Libraries**: Reusable code across apps
